# Weater-will-know-the-answer 水會知道答案的
#水面塔羅

遊玩方式: 誠心向水面發問一個是非問題，接著用手觸摸水面，水會回答YES、NO

-技術連接: ARDUINO觸摸感測器-->連接鱷魚夾-->銅片-->水面

arduino接線: 觸摸面板上有三個接腳，
gnd->接gnd
vcc-->接5v
sig-->接digital 2 pin

黑色鱷魚夾 夾觸摸感測器、紅色夾銅片

軟體串聯: ARDUINO數據傳輸到TOUCHDESIGNER
